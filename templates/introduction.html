Introduction to Temporal Reasoning Task

What is the Temporal Reasoning Task?

The Temporal Reasoning Task is a cognitive experiment designed to test how well participants can detect patterns that emerge over time. It challenges both human players and language models to identify statistical biases in color distributions across different spatial locations.

How does it work?

In each game: - You'll see multiple rounds with colored buttons - One quadrant is biased towards showing a particular color - Your task is to identify which quadrant has the bias - The bias only becomes apparent over multiple rounds
